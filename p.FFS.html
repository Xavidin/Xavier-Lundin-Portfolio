Fast Food Stock

Wanted to find the current prices and performance of fast food stocks.

  • Created a dashboard format where I could imput information
  • Inputed familiar fast food brand and converted to stock format in the Data section
  • Used Excel's Stock feature to input other values so it would consistently update
  • Used conditional formating to make table cells more visually appealing

    • At this point I realized I wanted to treat this like a real stock profile and track gains and loses. The main problem was that I did not have any personal information to use.

    To calculate for more personalized attributes I:

  • Generated a random amount of shares
  • Found the 52 week high and low using Excel's Stock feature
  • Generated a randomized user buy-in price using RANDBETWEEN function
    • Technologies: Excel
