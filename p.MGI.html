Movie Genre interactivity

As a Bachelor in Film and Media Studies, I was curious to see what Genres some of the top rated films were. This insight could be vital in the creation and advertisement of the next big film.

To start of this project I used a publically sourced dataset from kaggle featuring the top 5000 films on IMDb. From there I:
  • Cleaned and organized the data using Excel
  • Inputed my Excel sheet into RStudio
  • Explored the Tables more using functions
  • Added quick clean up in RStudio
  • Created tables and graphs to comunicate the data

    • From here I was able to graph and filter the data in order to get the information I set out for and was able to comunicate this information in a R Markdown;

    • Technologies: Power BI
    • RPubs: Movie Genre Interactivity
    • Dataset: Kaggle
